Utilisation d'un Jumbotron(cet element est un panel-succes).
Le jumbotron est un composant de Bootstrap. Il affiche un bloc situe en haut de la page d;un site. Ce bloc est d'attirer l'attention de l'utilisateur sur un element particulier. Ce bloc est gris par defaut, mais il est possible de le modifier grace au CSS et a JavaScript pour qu'il puisse contenir une image situee en arriere-plan.

Mise en place d'une pagination
We use a large block of connected links for our pagination, making links hard to miss and easily scalable--all while providing large hit areas. Pagination is bulit with list HTML elements so screen readers can announces the number of avaliable links, Use a wrapping "nav" element to identify it as a navigation section to screen readers and other assisitive technologies.
